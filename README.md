**Fokus - Pomodoro com To-Do List**

O Fokus é uma aplicação simples de Pomodoro com uma funcionalidade adicional de To-Do List, permitindo que você gerencie suas tarefas de maneira eficiente, concentrando-se em períodos de foco seguidos por pausas curtas e longas.
Funcionalidades

    Pomodoro temporizador: Configure o tempo de foco para suas tarefas e mantenha-se produtivo.
    Lista de A fazer: Adicione e gerencie suas tarefas pendentes.
    Pausas Personalizadas (Pausas Personalizadas): Defina pausas curtas e longas para otimizar sua produtividade.

**Como Usar**

    Clone este repositório em sua máquina local.



git clone https://github.com/miguelvaztostes/projeto_fokus.git
